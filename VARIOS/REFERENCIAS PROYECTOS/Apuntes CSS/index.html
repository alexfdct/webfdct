INTROUDCCIÓN A CSS (CASCADING STYLE SHEETS)


Para poder utilizar CSS en HTML, podemos hacerlo de tres maneras o formas:
  • CSS Externo: es aquel que va dentro de la cabecera de mi html (head).
    Tiene la siguiente forma: link rel (release) = stylesheet type=text/css href=""
  • CSS Interno: es aquel que va dentro de la cabecera de mi html (head), pero
    va entre las etiquetas style de aperta y de cierre.
  • CSS Embebido: es aquel que se aplica directamente a la etiqueta HTML

Elementos ID y Class utilizados en CSS para HTML

Elemento ID


El elemento ID, se puede definir en CSS y HTML como el atributo para un elemento único.
Es decir, no debería haber otro elemento con el mismo nombre de identificador (id)
dento de un mismo HTML.

Elemento CLASS


El elemento CLASS, se puede definir en CSS y HTML, a diferencia del valor
del atributo ID, como un elemento que puede ser utilizado en más de una etiqueta
de un documento HTML. Es decir, lo puedo usar en varios tipos de etiquetas distintas
