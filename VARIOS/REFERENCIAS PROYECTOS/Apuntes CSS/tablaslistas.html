Alineación Textos

Los textos en HTML, cuando queremos alienarlos con CSS, utilizamos la propiedad text-align.
Para ello, utilizamos los siguientes atributos:
  • Center: centrar texto
  • Jutify: justificación del texto
  • Right: para alinear texto a la derecha
  • Left: alinear texto a la izquierda

Propiedades CSS para Tablas

Las propiedes que podemos aplicar a las tablas en CSS, son las siguientes:
  • border-collapse: permite especificar si el borde de una tabla y sus celdas deben estar
    unidos (collapse) o separados (separate)
  • border-spacing: especifica el tamaño que medirán los espacio exteriores entre celdas
  • caption-side: permite especificar donde se colocará el título de la tabla (en el caso
    de haber usado caption): al principio de la tabla (top) o al final (bottom)
  • empty-cells: establecer si mostrar (show) u ocultar (hide) las celdas vacías, haciendo
    desaparecer estas últimas.
  • table-layout: permite especificar si el navegador debe adaptar el tamaño de las celdas
    automáticamente (auto) o establecer un tamaño fijo (fixed)

Listas en CSS: list-style-type

Los marcadores de listas ordenadas (ol) y de las listas desordenadas (ul), pueden ser modificables a través
del atributo list-style-type

Listas Desordenadas (ul)

  • disc
  • circle
  • square
  • none

Listas Ordenadas (ol)

  • decimal: 1,2,3...
  • decimal-leading-zero: 01,02,03...
  • lower-roman/upper-roman: i,ii,iii,iv... I,II,III,IV...
  • lowe-alpha/lower-latin y el upper-alpha/upper-latin:aA, bB, cC, dD, eE...

Listas en CSS: list-style-image
