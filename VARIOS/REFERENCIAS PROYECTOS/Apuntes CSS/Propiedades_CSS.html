Colores en CSS

En CSS podemos introducir los colores de 2 maneras distintas

  • Si vamos a introducir un color de texto: utilizamos la propiedad color
  • si vamos a introducir un color de fondo (body, divs, etc): utilizamos la propiedad
    background-color

Las maneras en las que podemos poner los colores en CSS son de las siguientes formas:

  • Palabra del color predefinida: Eje: red, blue, green, etc.
  • Esquema RGB: Red Green Blue en decimal (0-255). Eje: rgb (245,23,36)
  • Esquema RGB con un capa alpha. La capa alpha, representa
    la opacidad del color al que estamos aplicando el RGB.
    Estos valores van desde el 0 como el más transparente, y el 1 como el más opaco. Eje: rgb (245,23,36,0.5)
  • Esquema RGB: Red Green Blue (Hexadecimal):#RRGGBB. Ejemplo: #FF0022
  • Esquema RGB: Red Green Blue (Hexadecimal) pero abreviados:#RGB. Ejemplo: #F20223 quedaría de esta manera: #FF0022

Modelo de Cajas en CSS

Todos los elementos que vamos a utilizar en las páginas webs son cajas.
Vamos a poder representar en las cajas todos los elementos existentes en HTML: listas, cabeceras, etc.
Para ello vamos a utilizar una serie de propiedades a tener en cuenta como son las siguientes:
  • Margin: son los márgenes que vamos a tener entre el contenedor (caja) y el navegador
  • Padding: son los márgenes que vamos a tener entre el contenido de nuestra caja y los
    bordes de esta.

Zonas de un Elemento de una Caja

  • Top: se refiere a la parte superior del elemento
  • Left: se refiere a la parte izquierda del elemento
  • Right:se refiere a la parte derecha del elemento
  • Bottom:se refiere a la parte inferior del elemento
  • Center: se refiere a la posición central entre dos extremos
