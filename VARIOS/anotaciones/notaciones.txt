IDEAS

NOMBRE EMPRESA: Open System Consulting (OSC) "Opsycon"
El modelo de negocio/empresa sería de una "tienda" que se encarga de compar y vender equipos, además de restaurarlos
arreglarlos y de configurar equipos completamente bajo demanda.

Los servicios que prestarían son:
- Asistencia técnica: El cliente nos trae su equipo (Ya sea portátil o de sobremesa) y nosotros le haremos
el correspondiente diagnóstico y, si el cliente acepta el presupuesto de la operación, haremos las modificaciones
oportunas
- Cambio de componentes: Nos ofrecemos a cambiar los componentes de tu equipo por unos nuevos, ya sea bajo petición 
específica del cliente o bajo nuestra opinión.
- Compra/Venta de equipos de sobremesa: Compramos equipos de sobremesa de personas que no quieran utilizarlos, los 
restauramos y los vendemos.
Configuración de un nuevo equipo bajo demanda: Configuramos un equipo desde cero, bajo la demanda del cliente.

Colores secundarios:

#6f5643 (marrón oscuro): Este color se puede usar para los títulos, los subtítulos, los bordes o los elementos decorativos.
#cc6b49 (naranja rojizo): Este color se puede usar para los botones, los menús, las pestañas o los elementos que quieres que destaquen.
#d2a24c (amarillo mostaza): Este color se puede usar para los iconos, las imágenes o los elementos que quieres que sean llamativos.
Color de contraste:

#738da8 (azul grisáceo): Este color se puede usar para los enlaces, los botones de llamada a la acción o los elementos que quieres que destaquen.
Ejemplos de cómo usar la jerarquía de colores:

Encabezado: Usa el color principal (#f1f1f1) para el fondo del encabezado y el color secundario (#6f5643) para el texto del encabezado.
Cuerpo de la página: Usa el color principal (#f1f1f1) para el fondo del cuerpo de la página y el color secundario (#cc6b49) para los títulos y subtítulos.
